React 프로젝트 필수 라이브러리 12선

개요

현대 React 프로젝트에서 표준으로 사용되는 12개 라이브러리에 대한 설명과 실무 예시입니다. 타입 안전성, 개발 생산성, 성능 최적화에 중점을 둔 풀스택 구성입니다.

라이브러리 목록

  1. date-fns - 날짜/시간 처리
  2. ts-pattern - 타입 안전 패턴 매칭
  3. @tanstack/react-query - 서버 상태 관리
  4. zustand - 전역 상태 관리
  5. react-use - 커스텀 훅 모음
  6. es-toolkit - 유틸리티 함수
  7. lucide-react - 아이콘 라이브러리
  8. zod - 스키마 검증
  9. shadcn-ui - UI 컴포넌트
  10. tailwindcss - CSS 프레임워크
  11. supabase - BaaS
  12. react-hook-form - 폼 관리

1. date-fns

특징

  • 모듈식 구조로 필요한 함수만 import 가능 (트리쉐이킹)
  • moment.js보다 가벼움
  • 불변성 유지 (원본 날짜 객체 변경 안 함)

주요 기능

import { format, addDays, differenceInDays, isAfter } from 'date-fns';
import { ko } from 'date-fns/locale';

// 날짜 포맷팅
const now = new Date();
format(now, 'yyyy-MM-dd HH:mm:ss'); // "2025-12-22 14:30:00"
format(now, 'PPP', { locale: ko }); // "2025년 12월 22일"

// 날짜 계산
const nextWeek = addDays(now, 7);
const daysUntil = differenceInDays(nextWeek, now); // 7

// 날짜 비교
const deadline = new Date('2025-12-31');
isAfter(deadline, now); // true

2. ts-pattern

특징

  • switch문을 타입 안전하게 대체
  • exhaustive checking (모든 케이스 처리 강제)
  • 복잡한 조건 분기를 명확하게 표현

주요 기능

import { match, P } from 'ts-pattern';

type State = 
  | { status: 'idle' }
  | { status: 'loading' }
  | { status: 'success'; data: string }
  | { status: 'error'; error: Error };

function handleState(state: State) {
  return match(state)
    .with({ status: 'idle' }, () => 'Ready to start')
    .with({ status: 'loading' }, () => 'Loading...')
    .with({ status: 'success' }, ({ data }) => `Got: ${data}`)
    .with({ status: 'error' }, ({ error }) => `Error: ${error.message}`)
    .exhaustive(); // 모든 케이스를 처리하지 않으면 타입 에러
}

// 복잡한 조건
match(value)
  .with(P.number, (n) => `숫자: ${n}`)
  .with(P.string, (s) => `문자열: ${s}`)
  .with({ age: P.number.gte(18) }, () => '성인')
  .otherwise(() => '기타');

3. @tanstack/react-query

특징

  • 서버 상태 관리 전문
  • 자동 캐싱, refetch, 동기화
  • 낙관적 업데이트 지원

주요 기능

import { useQuery, useMutation, useQueryClient } from '@tanstack/react-query';

// 데이터 조회
function UserProfile({ userId }: { userId: string }) {
  const { data, isLoading, error } = useQuery({
    queryKey: ['user', userId],
    queryFn: () => fetch(`/api/users/${userId}`).then(r => r.json()),
    staleTime: 1000 * 60 * 5, // 5분간 캐시 유지
  });

  if (isLoading) return <div>Loading...</div>;
  if (error) return <div>Error!</div>;
  return <div>{data.name}</div>;
}

// 데이터 변경
function UpdateUser() {
  const queryClient = useQueryClient();
  
  const mutation = useMutation({
    mutationFn: (newUser: User) => 
      fetch('/api/users', { method: 'POST', body: JSON.stringify(newUser) }),
    onSuccess: () => {
      queryClient.invalidateQueries({ queryKey: ['user'] });
    },
  });

  return (
    <button onClick={() => mutation.mutate({ name: 'John' })}>
      Update
    </button>
  );
}

4. zustand

특징

  • Redux보다 간단한 보일러플레이트
  • React 외부에서도 사용 가능
  • 미들웨어 지원 (persist, devtools)

주요 기능

import { create } from 'zustand';

// 스토어 정의
interface BearStore {
  bears: number;
  increase: () => void;
  decrease: () => void;
  reset: () => void;
}

const useBearStore = create<BearStore>((set) => ({
  bears: 0,
  increase: () => set((state) => ({ bears: state.bears + 1 })),
  decrease: () => set((state) => ({ bears: state.bears - 1 })),
  reset: () => set({ bears: 0 }),
}));

// 컴포넌트에서 사용
function BearCounter() {
  const bears = useBearStore((state) => state.bears);
  return <h1>{bears} bears</h1>;
}

function Controls() {
  const { increase, decrease, reset } = useBearStore();
  return (
    <>
      <button onClick={increase}>+</button>
      <button onClick={decrease}>-</button>
      <button onClick={reset}>Reset</button>
    </>
  );
}

// persist 미들웨어
import { persist } from 'zustand/middleware';

const useStore = create(
  persist(
    (set) => ({ count: 0, inc: () => set((s) => ({ count: s.count + 1 })) }),
    { name: 'my-storage' } // localStorage에 저장
  )
);

5. react-use

특징

  • 자주 사용하는 커스텀 훅 제공
  • 보일러플레이트 감소

주요 기능

import { 
  useLocalStorage, 
  useDebounce, 
  useToggle,
  useWindowSize,
  useAsync,
} from 'react-use';

function Example() {
  // localStorage 동기화
  const [theme, setTheme] = useLocalStorage('theme', 'dark');
  
  // 디바운스
  const [searchTerm, setSearchTerm] = useState('');
  const debouncedSearch = useDebounce(searchTerm, 500);

    </div>
  );
}

6. es-toolkit

특징

  • lodash 대체재
  • 성능 최적화
  • TypeScript 친화적

주요 기능

import { debounce, throttle, chunk, groupBy, pick, omit } from 'es-toolkit';

// 디바운스/쓰로틀
const handleSearch = debounce((term: string) => {
  console.log('Searching:', term);
}, 300);

const handleScroll = throttle(() => {
  console.log('Scrolling...');
}, 100);

// 배열 조작
const items = [1, 2, 3, 4, 5, 6, 7, 8];
chunk(items, 3); // [[1,2,3], [4,5,6], [7,8]]

const users = [
  { name: 'Alice', role: 'admin' },
  { name: 'Bob', role: 'user' },
  { name: 'Charlie', role: 'admin' },
];
groupBy(users, (u) => u.role);
// { admin: [...], user: [...] }

// 객체 조작
const user = { id: 1, name: 'John', email: 'john@ex.com', password: '123' };
pick(user, ['id', 'name']); // { id: 1, name: 'John' }
omit(user, ['password']); // { id: 1, name: 'John', email: '...' }

8. zod

특징

  • 스키마 검증 + TypeScript 타입 추론
  • 런타임 타입 안전성
  • transform 지원

주요 기능

import { z } from 'zod';

// 스키마 정의
const UserSchema = z.object({
  id: z.string().uuid(),
  email: z.string().email(),
  age: z.number().min(18).max(120),
  role: z.enum(['admin', 'user', 'guest']),
  profile: z.object({
    name: z.string().min(2),
    bio: z.string().optional(),
  }),
  tags: z.array(z.string()),
});

// TypeScript 타입 추론
type User = z.infer<typeof UserSchema>;

// 검증
const result = UserSchema.safeParse({
  id: '123e4567-e89b-12d3-a456-426614174000',
  email: 'user@example.com',
  age: 25,
  role: 'user',
  profile: { name: 'John' },
  tags: ['developer'],
});

if (result.success) {
  console.log(result.data); // 타입: User
} else {
  console.error(result.error.issues);
}

// API 응답 검증
async function fetchUser(id: string): Promise<User> {
  const response = await fetch(`/api/users/${id}`);
  const data = await response.json();
  return UserSchema.parse(data); // 실패시 throw
}

// transform
const DateSchema = z.string().transform((str) => new Date(str));

import { createClient } from '@supabase/supabase-js';

const supabase = createClient(
  'https://xxx.supabase.co',
  'your-anon-key'
);

const { data, error } = await supabase
  .from('users')
  .select('*')
  .eq('status', 'active')
  .order('created_at', { ascending: false })
  .limit(10);

12. react-hook-form

특징

  • 리렌더 최소화
  • zod와 완벽한 통합
  • 비제어 컴포넌트 방식

주요 기능

import { useForm } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';

const schema = z.object({
  email: z.string().email(),
  password: z.string().min(8),
  age: z.number().min(18),
});

type FormData = z.infer<typeof schema>;

function LoginForm() {
  const {
    register,
    handleSubmit,
    formState: { errors, isSubmitting },
    watch,
    setValue,
    reset,
  } = useForm<FormData>({
    resolver: zodResolver(schema),
    defaultValues: {
      email: '',
      password: '',
      age: 18,
    },
  });

  const onSubmit = async (data: FormData) => {
    await fetch('/api/login', {
      method: 'POST',
      body: JSON.stringify(data),
    });
  };

  // 특정 필드 감시
  const emailValue = watch('email');

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <div>
        <input {...register('email')} placeholder="Email" />
        {errors.email && <span>{errors.email.message}</span>}
      </div>

      <div>
        <input 
          {...register('password')} 
          type="password" 
          placeholder="Password" 
        />
        {errors.password && <span>{errors.password.message}</span>}
      </div>

      <div>
        <input 
          {...register('age', { valueAsNumber: true })} 
          type="number" 
        />
        {errors.age && <span>{errors.age.message}</span>}
      </div>

      <button type="submit" disabled={isSubmitting}>
        {isSubmitting ? 'Submitting...' : 'Submit'}
      </button>
      
      <button type="button" onClick={() => reset()}>
        Reset
      </button>
    </form>
  );
}

// 중첩된 필드
const nestedSchema = z.object({
  user: z.object({
    firstName: z.string(),
    lastName: z.string(),
  }),
  addresses: z.array(z.object({
    street: z.string(),
    city: z.string(),
  })),
});

function NestedForm() {
  const { register } = useForm<z.infer<typeof nestedSchema>>({
    resolver: zodResolver(nestedSchema),
  });

  return (
    <form>
      <input {...register('user.firstName')} />
      <input {...register('user.lastName')} />
      <input {...register('addresses.0.street')} />
      <input {...register('addresses.0.city')} />
    </form>
  );
}

스택 구성의 특징

  1. 타입 안전성 강조

    • TypeScript 친화적인 라이브러리 선택
    • zod로 런타임 검증 + 타입 추론
    • ts-pattern으로 exhaustive checking

  2. 상태 관리 분리

    • 서버 상태: React Query
    • 전역 상태: zustand
    • 폼 상태: react-hook-form

  3. 개발 생산성

    • shadcn-ui로 빠른 UI 구성
    • react-use로 보일러플레이트 감소
    • es-toolkit로 유틸리티 함수 제공

  4. 성능 최적화

    • 트리쉐이킹 지원 라이브러리
