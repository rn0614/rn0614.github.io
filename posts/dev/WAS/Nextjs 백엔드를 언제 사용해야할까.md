# Nextjs 백엔드
> nextjs 에서 api/ route.ts 를 이용해 주소를 받는 백엔드 동작을 하는 위치

# 언제 사용할까?
우선 기본적으로는 Nextjs 백엔드의 경우 작은 서버일때 동시에 이용할것. 규모가 커진 서버에서 서버 부하에 따라서 서버를 증설하는데 back/ front 부분을 따로따로 증설이 가능함. api router를 쓰면 두개를 동시에 증설해야해서 낭비가 된다.
