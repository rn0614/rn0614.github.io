# 개요
`<></>` 이 React.Fragment의 축약어이다.

단 list의 key값이나 다른 props를 넘기려면 축약어가 아닌 상태로 사용해야한다.
```tsx
itemList.map((todo, idx)=>{
  <React.Fragment key={idx}>
    <td></td>
    <td></td>
  </React.Fragment>
})

```
