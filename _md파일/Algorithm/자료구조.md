#### Stack
```js
class Stack {
  arr =[];

  push(value){
    return this.arr.push(value);  // push의 return 값은 arr.length
  }
  pop(){
    return this.arr.pop();
  }

  top(){
    return this.arr.at(-1);
  }

  get length(){
    return this.arr.length;
  }
}

```


#### Queue
> 선입선출 과정인데 이때 arr.shift는 O(n) 의 시간복잡도를 갖는다. 이를 개선하기 위해서는 arr를 LinkedList로 변경하고 tail부의 push, head부의 shift가 시간복잡도 O(1)을 갖도록 개선이 필요하다.
```js
class Queue {
  arr =[];

  enqueue(value){
    return this.arr.push(value);  // push의 return 값은 arr.length
  }
  dequeue(){
    return this.arr.shift();
  }

  peak(){
    return this.arr[0];
  }

  get length(){
    return this.arr.length;
  }
}

```


#### Tree

```js
class Tree{
  constructor(value){
    this.root = new Node(value);
  }
}

class Node {
  children =[];
  constructor(value){
    this.value = value
  }

  push(value){
    this.children.push(new Node(value));
  }
}

const tree = new Tree(50);
tree.root.push(29);
tree.root.push(75);

tree.root.children[0].push(12)
```

##### 이진 트리
1. Full : 자식노드가 0 or 2
2. Perfect : 완전한 삼각형 모양
3. Complete : 왼쪽부터 순서대로 채워지는 방식
4. Degenerate : 자식노드가 0 or 1 (안좋은 구조)
5. Balanced : 모든 노드가 dif가 0, 1 인경우     -> O(log N)을 보장하는 방식
