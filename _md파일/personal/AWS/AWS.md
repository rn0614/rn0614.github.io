### EC2(Amazon EC2)
가상의 컴퓨터를 빌려쓰는 서비스 - 클라우드 컴퓨팅

### IAM
aws의 보안 및 관리를 담당하는 서비스
- 금액관리
- 권한
- 사용자 생성, 관리, 계정보안 등

### VPC
aws계정전용 가상네트워크 (가상으로 존재하는 네틍워크) - private 서비스
다양한 서브넷 구성
보안설정

### S3
객체 스토리지 서비스 ( 파일 저장소 )
