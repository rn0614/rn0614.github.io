# S3 관련 작업
> S3 bucket을 만들고 파일을 업로드 하는 2가지 방식
### 1. 수동 생성방식
aws 에서 S3 들어가서 수동 버킷 생성 해당 버킷에 들어가서 파일 업로드 방식

### 2. console 업로드 방식(AWS CLI 이용)
연결된 EC2 입장
region이 맞아야 제대로 생성됨

1. EC2 Session Manager 접근

```shell
// 해당 인스턴스가 소유한 버킷 목록 검색
aws s3 ls

// 해당 인스턴스의 버킷을 생성
aws s3 mb s3://[버킷명칭]

// 인스턴스 to bucket 파일 전송
aws s3 cp /home/ssm-user/[파일명] s3://[버킷명칭]

// 버킷에서 소유한 객체 목록 표기
aws s3 ls s3://[버킷명칭]

```



서브넷은 한 번에 하나의 라우팅 테이블에만 연결할 수 있지만, 여러 서브넷을 같은 라우팅 테이블에 연결할 수 있습니다.


DNS란? : 인터넷에서 도메인 이름을 IP주소로 변환해주는 시스템.
- 도메인 이름/ IP주소/ DNS서버 로 구성됨

VPC란? : 가상 네트워크 IP주소, 서브넷, 라우팅테이블, IGW, 보안그룹 등을 조정함

서브넷이란? : IP 범위 대역을 나눈 조각/ 들어온 연결에 대해 라우팅 테이블 대열역대로 나눠져 각자를 실행
